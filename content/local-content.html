本地内容示例

这是一个通过客户端JavaScript加载的本地HTML文件示例。

功能特点

  • 使用fetch API加载本地内容
  • 无需刷新页面即可更新内容
  • 支持错误处理和加载状态
  • 响应式设计，适配不同屏幕尺寸

代码示例

// 使用fetch加载本地内容 const loadLocalContent = async (url) => { setIsLoading(true); setContentError(''); try { const response = await fetch(url); if (!response.ok) { throw new Error(`HTTP error! status: ${response.status}`); } const htmlContent = await response.text(); setContent(htmlContent); } catch (error) { console.error('加载内容失败:', error); setContentError('加载内容失败，请稍后重试'); setContent(''); } finally { setIsLoading(false); } };

注意事项

本地内容加载的优点：

  • 避免了iframe的性能开销
  • 保持页面的一致性和美观
  • 可以与主应用进行更好的交互
  • 避免了跨域问题
